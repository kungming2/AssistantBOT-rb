*Note: This wiki page serves as a guide to the different sections of the daily statistics pages that Artemis generates, with explanations on what they are/mean.*

## Devvit relevance

The current Devvit version generates a smaller, installation-scoped statistics page than the legacy Python bot. Legacy statistics pages are preserved at `assistantbot_statistics_legacy`, but the active Devvit page at `assistantbot_statistics` only includes data the Devvit app can collect from Reddit triggers, scheduled jobs, and Redis. Old statistics sections that are no longer generated are listed in [Deprecated Python behavior](deprecated.md#statistics).

Legend:

* **Current**: present on the Devvit statistics page.
* **Changed**: the same general idea exists, but the Devvit page renders it differently.

| 20xx-xx-xx | XXX | +XX |

*↑ Explanation: The active Devvit page displays the most recent 90 subscriber snapshots and the change from the previous stored snapshot.*

### Subscriber trends

**Changed.**

The old standalone averages are replaced by the `Overall` section's 7-day and 30-day subscriber trend table. Older subscriber trend fields are listed in [Deprecated Python behavior](deprecated.md#statistics).

### Subscriber milestones

**Current, starting from Devvit snapshots.**

*Next Subscriber Milestone (est.)*: 4000 subscribers on 2019-01-15 (6 days from now).

*↑ Explanation: The Devvit version estimates the next subscriber milestone from the recent subscriber snapshots it has recorded. The estimate uses the average daily change over the latest snapshot sample and is hidden when the subreddit is shrinking, when there is not enough snapshot history, or when the next milestone is too far away.*

*The milestone list starts at 10 subscribers and continues through major rounded subscriber counts. Artemis only projects from snapshots collected by this Devvit installation.*

The old milestone-history table is listed in [Deprecated Python behavior](deprecated.md#statistics). The active page only shows the next forward-looking milestone projection when recent subscriber snapshots support one.

## Traffic

Traffic data is not collected by the Devvit port because Devvit does not expose an equivalent subreddit traffic endpoint. See [Deprecated Python behavior](deprecated.md#statistics).

## Userflairs (optional)

**Changed.**

The legacy standalone userflair section has moved into `Overall` as `User Flair Distribution`. Older userflair fields are listed in [Deprecated Python behavior](deprecated.md#statistics).
